ContributingÂ¶

We welcome anyone to make contributions to this project. Just fork the develop branch of our GitHub Repository, make your changes, then issue a pull request on the develop branch.

Any pull request you make will need to have associated tests with them. If a test is not provided, the pull request will be closed and not merged. Also, any pull requests made to a branch other than develop will be closed and a new submission will need to be made to the develop branch.
