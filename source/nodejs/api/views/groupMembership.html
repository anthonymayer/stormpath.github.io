GroupMembership

A GroupMembership resource represents the association between an Account and a Group. When an Account is added to a Group or a Group is assigned to an Account, a GroupMembership is created.

Since: 0.1

method getAccount([options,] callback)

Retrieves the membership's associated Account and provides it to the specified callback.

Usage

membership.getAccount(function(err, account) {
  console.log(account);
});

You can also use resource expansion options (query params) to obtain linked resources in the same request:

membership.getAccount({ expand:'groups' }, function(err, account) {
  console.log(account);
});

Parameters

ParameterTypePresenceDescription
optionsobjectoptionalName/value pairs to use as query parameters, for example, for resource expansion.
callbackfunctionrequiredThe callback to execute upon resource retrieval. The 1st parameter is an Error object. The 2nd parameter is the retrieved Account resource.

Returns

void; the retrieved Account resource will be provided to the callback as the callback's second parameter.
