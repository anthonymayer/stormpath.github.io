AssertionAuthenticationResult

An AsssertionAuthenticationResult is returned by the StormpathAssertionAuthenticator. It encapsulates an authentication result from ID Site Callback or SAML Callback, and allows you to get the account that has authenticated.

Since: 0.16.0

method getAccount([options,] callback)

Retrieves the Account object of the user that has authenticated.

Usage

assertionAuthenticationResult.getAccount(function(err, account) {
  console.log(account);
});

Parameters

ParameterTypePresenceDescription
optionsobjectoptionalName/value pairs to use as query parameters, for example, for resource expansion.
callbackfunctionrequiredThe callback to execute upon resource retrieval. The 1st parameter is an Error object. The 2nd parameter is the retrieved Account resource.

Returns

void; the retrieved Account resource will be provided to the callback as the callback's second parameter.

property .stormpath_token String

The original JWT that was returned to your application, as the ?jwtResponse=<stormpath_token> parameter.

property .expandedJwt Object

The parsed stormpath_token.
